# Plataforma de Análise de Futebol

Site de análise inteligente para apostas esportivas, focado em estatísticas ao vivo, probabilidades e identificação das melhores oportunidades no mercado de futebol.

## Funcionalidades
- Análise em tempo real de jogos
- Estatísticas de gols, escanteios, cartões e posse de bola
- Sugestões de apostas com base em dados
- Interface rápida e responsiva
- Foco em apostas seguras e estratégicas

## Tecnologias
- React + Vite
- API de dados esportivos
- Deploy na Vercel

## Objetivo
Ajudar apostadores a tomarem decisões mais assertivas utilizando dados e inteligência, transformando informação em vantagem.
